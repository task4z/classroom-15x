# [CLASSROOM15X](https://classroom15x.net)

### Open-Source Web Development Learning Modules & Interactive Resources

Welcome to the **classroom-15x** repository. This project serves as a centralized hub for browser-based interactive modules, designed to test hardware acceleration, input latency, and UI/UX responsiveness in educational environments.

## 📚 Project Overview
This repository contains a collection of **169 interactive lessons** and resource modules. These assets are structured to assist educators and students in understanding web-based logic and interactive media performance through browser-based simulations.

To ensure compatibility with high-security educational networks and automated indexing, all assets follow a standardized naming convention:

* **Resource ID:** `lesson001` through `lesson169`
* **Format:** HTML5 / JavaScript / WebGL standard deployment.
* **Hosting:** Optimized for GitHub Pages.

## 🛠 Technical Specifications

- **Core Framework**: HTML5 / CSS3 / Vanilla JavaScript  
- **Graphics Engine**: WebGL / Canvas API for performance testing.  
- **Optimization**: Minified assets for low-bandwidth environments.  
- **Compatibility**: Fully tested on ChromeOS (Chromebooks), Windows, macOS, and Linux.
